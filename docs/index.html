Mol framework

Librería de estilos CSS modular

mol = normalize + core + grid

Mol es una librería de CSS cuyo objetivo principal es, además economizar el tiempo de desarrollo y de carga de las hojas de estilos, agregar la habilidad de editar fácilmente las clases con archivos de variables para evitar que todas las páginas parezcan ser la misma.

Cada módulo pueda utilizarse solo, o bien pueden cargarse todos los módulos desde un único archivo. Las caracteríasticas generales del estilo son manipulables mediante archivos de variables en SCSS esperando con ésto reducir el esfuerzo para llegar al diseño final de cada proyecto.

npm install --save-dev https://github.com/MolFramework/mol.git
